# 游깷 SasyaMarg Frontend

This is the **frontend application** for **SasyaMarg**, built using **React (Vite)**.

The frontend handles:
- User interfaces for Farmer, Buyer, and Admin
- Role-based navigation
- API integration with the backend
- Wishlist and discovery flows
